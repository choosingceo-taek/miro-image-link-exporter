# RACK 상품 수집기 (Chrome 확장)

브랜드/카테고리 **URL 목록**을 넣으면, 확장이 각 페이지를 **백그라운드 탭으로 자동 방문**해
스크롤·수집한 뒤 **RACK 공유 저장소(미로 앱)** 로 전송합니다. 실제 크롬 탭이라 **Akamai 등 봇 차단을
자연히 통과**합니다. (Zara·Massimo Dutti·H&M·Gap 등)

> 서버가 아니라 **본인 크롬**에서 돌기 때문에 무료이며, 크롬만 켜져 있으면 됩니다.
> 완전 무인(서버 자동)은 유료 헤드리스 브라우저가 필요합니다.

## 설치 (팀원 각자, 최초 1회)
1. 이 폴더(`chrome-extension`)를 내려받습니다. (저장소에서 이 폴더만 zip으로 받아 풀어도 됨)
2. 크롬 주소창에 `chrome://extensions` 입력
3. 우상단 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램을 로드** → 이 `chrome-extension` 폴더 선택
5. 툴바에 **RACK 수집기** 아이콘이 생김. (Worker 주소·토큰은 기본 내장 — 설정 불필요)

## 사용
1. 아이콘 클릭 → 팝업 열기
2. **브랜드/카테고리 URL**을 한 줄에 하나씩 붙여넣기 (각 사이트의 **상품 목록 페이지** 주소)
   - 예: `https://www.zara.com/kr/ko/woman-new-in-l1180.html`
3. **수집 시작** → 확장이 URL을 하나씩 백그라운드 탭으로 열어 자동 수집·전송, 진행률·로그 표시
4. 끝나면 미로 앱 **Market Research → 저장된 카탈로그** 칩에 자동으로 뜸 (팀 전체 공유)

## 참고
- 한 번에 여러 개 넣으면 사이트 부담·차단 방지를 위해 **하나씩 텀을 두고** 처리합니다(개당 수 초~십수 초).
- "상품 못 찾음"이 뜨면: 그 URL이 상품 목록이 아니거나(상세 페이지 등), 사이트 구조가 특이한 경우입니다.
  그 URL을 알려주시면 수집 로직을 보강합니다.
- Worker 주소/토큰을 바꾸려면 팝업 하단 **⚙ Worker 설정**에서 변경.
