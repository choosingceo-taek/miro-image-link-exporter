# 📘 이미지 링크 추출기 — 팀 사용 매뉴얼

미로 보드의 **사진 + 사진에 걸린 링크(URL)** 를 엑셀(CSV)로 뽑는 앱입니다.
이 매뉴얼은 **① 관리자(최초 1회 세팅)** 와 **② 팀원(설치·사용)** 두 부분으로 나뉩니다.

- **관리자(예: 나 한 명)** → `PART A` 를 딱 한 번만.
- **모든 팀원** → `PART B`(설치, 1회) + `PART C`(사용, 매번).

> 팀원은 코드나 서버를 몰라도 됩니다. **링크 클릭 → 설치 → 버튼 클릭**이 전부입니다.

---

# PART A. 관리자 세팅 (최초 1회, 약 10분)

이 앱은 이미 **공개 저장소 `choosingceo-taek/miro-image-link-exporter`** 에 올라가 있습니다.
남은 일은 (1) GitHub Pages 켜기 → (2) 미로에 앱 등록 → (3) 팀원용 설치 링크 공유 입니다.

## A-1. GitHub Pages 켜기 (앱을 인터넷 주소로 만들기)

- [ ] 이 저장소 → **Settings → Pages** 이동
- [ ] **Build and deployment → Source** 를 **`Deploy from a branch`** 로 설정
- [ ] **Branch** 를 **`main`** / 폴더 **`/ (root)`** 선택 후 **Save**
- [ ] 1~2분 뒤 페이지 상단에 표시되는 주소가 앱 주소입니다:

  ```
  https://choosingceo-taek.github.io/miro-image-link-exporter/index.html
  ```

- [ ] 위 주소를 브라우저에서 열어 **빈 흰 화면(정상)** 이 뜨는지 확인 (미로 밖에서는 UI가 안 보이는 게 정상입니다).

## A-2. 미로에 앱 등록하기

- [ ] https://miro.com/app/settings/user-profile/apps 접속 → **Create new app**
- [ ] 앱 이름 입력 (예: `이미지 링크 추출기`)
- [ ] **팀 선택**: 팀원들이 실제로 쓰는 **공용 팀** 선택 (중요 — 이 팀 안에서 공유됩니다)
- [ ] **Create app** 클릭

이어서 앱 설정 화면에서:

- [ ] **App URL** 칸에 A-1의 주소 입력:
      `https://choosingceo-taek.github.io/miro-image-link-exporter/index.html`
- [ ] **Permissions (Scopes)** 에서 **`boards:read`** 체크 후 저장
- [ ] **Web SDK** 사용 옵션이 있으면 켜기(활성화)
- [ ] 화면 아래 **Install app and get OAuth token** → 공용 팀에 **Install**

## A-3. 팀원용 설치 링크 복사

- [ ] 앱 설정 페이지 맨 아래 **Share app** 섹션으로 스크롤
- [ ] **Installation URL** (설치 링크)을 **복사**
- [ ] 이 링크를 팀 채팅에 **아래 PART B 안내와 함께** 공유

✅ 여기까지 하면 관리자 세팅 끝.

---

# PART B. 팀원 — 앱 설치 (최초 1회, 30초)

> 관리자가 공유한 **설치 링크(Installation URL)** 가 있어야 합니다.

- [ ] 관리자가 준 **설치 링크**를 브라우저 주소창에 붙여넣고 **Enter**
- [ ] 뜨는 설치/권한 창에서 **우리 공용 팀** 선택
- [ ] **Authorize(허용)** 클릭

✅ 끝. 한 명이 팀에 설치하면 같은 팀원 모두가 쓸 수 있습니다.

---

# PART C. 팀원 — 사용법 (필요할 때마다)

- [ ] 링크를 추출할 **미로 보드**를 엽니다.
- [ ] 왼쪽 도구 막대 맨 아래 **앱 아이콘** 에서 **이미지 링크 추출기** 클릭 → 오른쪽 패널이 열립니다.
- [ ] **[사진 링크 추출하기]** 클릭 → "전체 N개 중 링크 걸린 사진 M개" 요약과 미리보기 확인
- [ ] **[⬇︎ CSV 다운로드]** 클릭
- [ ] 내려받은 `miro_image_links_날짜.csv` 를 더블클릭 → **엑셀에서 바로 열림** (한글 안 깨짐)

📄 **엑셀 열 구성**

| 번호 | 제목 | 링크(하이퍼링크) | 이미지원본URL | 아이템ID |
|------|------|------------------|----------------|----------|
| 순번 | 사진 제목 | 사진에 걸어둔 링크 | 사진 파일 자체 주소 | 미로 내부 ID |

---

# ❓ 문제 해결

| 증상 | 확인 / 해결 |
|------|-------------|
| 왼쪽에 앱이 안 보임 | PART B 설치 링크를 다시 클릭. 그래도 없으면 새로고침(F5). |
| 추출 결과가 "0개" | 링크가 사진이 아니라 **도형·카드·텍스트**에 걸려 있을 수 있음(이 앱은 사진만 수집). |
| 링크 칸이 비어 있음 | 그 사진엔 실제로 링크가 안 걸린 것. `링크 없음`은 정상. |
| 다운로드 안 됨 | 브라우저 팝업/다운로드 차단 해제 후 패널 새로고침. |
| App URL 변경 시 | 호스팅 주소가 바뀌면 A-2의 App URL만 교체(팀원 재설치 불필요, 자동 반영). |

---

## 용어 한 줄 정리
- **GitHub Pages**: 저장소의 `index.html`을 인터넷 주소로 띄워주는 무료 호스팅.
- **설치 링크(Installation URL)**: 팀원이 누르면 미로에 앱이 추가됨.
- **`linkedTo`(하이퍼링크)**: 사진 클릭 시 이동하는 링크. 이 값 때문에 REST가 아닌 앱으로 만든 것.
