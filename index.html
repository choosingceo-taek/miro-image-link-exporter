자동으로 막히는 사이트는 이렇게 갱신합니다 (한 사람이 가끔, ~10초):

  1. 쇼핑몰 상품목록 페이지를 끝까지 스크롤 → F12 → Console (경고 시 allow pasting)
  2. 아래 📋 콘솔 코드 복사 → 콘솔에 붙여넣고 Enter (상품 자동 복사됨)
  3. 이 칸에 Ctrl+V → 미리보기에 추가 → 잡음 카드 체크 해제
  4. 보드에 추가 또는 ☁ 팀 저장소에 저장(팀 공유)

‘선택한 것만’은 보드에서 이미지를 클릭 · Shift+클릭 · 마우스 박스 드래그로 고른 뒤 누르세요.
